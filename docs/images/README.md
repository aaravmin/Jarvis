# README screenshots

The root [`README.md`](../../README.md) references the PNGs below. Until you drop them in here, GitHub
shows a broken-image icon in their place — so add at least `home.png`.

Capture them while signed in locally (`npm run dev` → http://localhost:3000). On macOS, `⌘⇧4` then
`Space` grabs a single window. Aim for a wide window (~1440px) and dark mode so the orb pops.

| File | Page | What to frame |
|------|------|---------------|
| **`home.png`** ⭐ | `/jarvis` (the home redirect) | The orb + the big military clock + the ask bar, hamburger closed. This is the hero — capture it first. |
| `jarvis-ask.png` | `/jarvis` | Mid-conversation: a question asked and the answer in the console, ideally showing web-search citations / a file it read. |
| `review.png` | `/review` | The Review queue with a couple of suggested items, each showing its **source chip**. |
| `contacts.png` | `/people` | One good contact card — background, relevance, the ask, and an AI-drafted outreach email. |
| `opportunities.png` | `/opportunities` | A few discovered opportunities with their resolved deadlines and source links. |

Keep them reasonably compressed (< ~500 KB each). Names must match exactly — the README links are
case-sensitive.

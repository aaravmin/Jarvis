# README screenshots

The root [`README.md`](../../README.md) references the PNGs below — and **only** these three, all of
which are already committed here. (Capture more later if you like, but the README no longer requires
any beyond these.)

Capture them while signed in locally (`npm run dev` → http://localhost:3000). On macOS, `⌘⇧4` then
`Space` grabs a single window. Aim for a wide window (~1440px) and dark mode so the orb pops.

| File | Page | What to frame |
|------|------|---------------|
| **`home.png`** ⭐ | `/jarvis` (the home redirect) | The orb + the big military clock + the ask bar, hamburger closed. This is the hero. |
| `jarvis-ask.png` | `/jarvis` | Mid-conversation: a question asked and the answer in the console, ideally showing web-search citations / a file it read. |
| `opportunities.png` | `/opportunities` | Discovered opportunities, each as its own card with deadline, location, requirements, and an application link. |

Keep them reasonably compressed (< ~500 KB each). Names must match exactly — the README links are
case-sensitive.
